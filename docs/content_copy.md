# Companion-Facing Copy Fragments

## Hero

江西东线旅行计划：从南昌进入江西，先上庐山，再去景德镇，最后用三清山作为自然风景压轴。整体路线基本固定，目前主要需要一起决定的是住宿选择、三清山东部/南部入口，以及最后一天是否直接夜间回上海。

## Route logic

这条路线的节奏是“城市缓冲 + 山上游览 + 陶瓷文化 + 自然压轴”。南昌和景德镇相对轻松，庐山和三清山消耗体力更多，所以住宿舒适度和天气备选方案比较重要。

## Lushan

庐山建议住在牯岭镇/牯岭街附近。西线适合作为上山后的轻松半日路线，东线则更经典也更累：含鄱口、五老峰、三叠泉需要看天气和体力决定是否完整走完。

## Jingdezhen

景德镇重点不是赶景点，而是陶瓷文化和街区体验。住宿优先考虑陶溪川附近，方便晚上逛街、市集和小店；如果想更老城氛围，可以看陶阳里附近。

## Sanqingshan

三清山是最后的自然风景压轴。一日游需要严格控制索道、大巴和返程高铁时间。东部金沙索道更适合一日游效率和性价比，南部外双溪配套可能更成熟。
