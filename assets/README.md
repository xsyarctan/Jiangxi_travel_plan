# Assets

Actual Notion image binaries were not downloaded into this bundle.

## Add these manually

### Route sketches

Place these files:

```text
public/sketches/lushan_route_sketch.png
public/sketches/sanqingshan_route_sketch.png
```

### Lodging photos

Use `data/lodging_options.json` field `image_refs` as expected local filenames.

Suggested location:

```text
public/images/lodging/
```

Example:

```text
public/images/lodging/IMG_7076.jpg
public/images/lodging/IMG_7049.jpg
...
```

The website should show placeholders when an image is missing.
